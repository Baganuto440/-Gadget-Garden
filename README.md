# Gadget Garden — сервис ремонта техники

## Описание проекта

**Gadget Garden** — это веб-приложение для сервисного центра по ремонту электроники. Проект выполнен в минималистичном стиле с использованием только чёрного, белого и зелёного цветов.

### Основной функционал:
- Просмотр информации о сервисном центре (адрес, часы работы, контакты)
- Просмотр списка мастеров с их контактами и специализацией
- Оставление заявки на ремонт через встроенную форму
- Сохранение заявок в базу данных SQLite
- Адаптивный дизайн для всех устройств

### Технологический стек:
| Компонент | Технология |
|-----------|------------|
| Фронтенд | HTML5, CSS3, JavaScript (Vanilla) |
| Бэкенд | Node.js, Express |
| База данных | SQLite |
| Стилизация | CSS Modules (5 файлов: base, layout, components, main, responsive) |

---

### Краткое содержание ТЗ:
1. Разработать одностраничный сайт для сервисного центра
2. Цветовая схема: чёрный (#000000), белый (#FFFFFF), зелёный (#2C7A3E)
3. На главной странице отобразить: услуги, мастеров с телефонами, адрес, форму заявки
4. Реализовать отправку заявок с сохранением в базу данных
5. Обеспечить адаптивность под мобильные устройства

### Инструкция входа

# Перейдите в папку проекта
cd gadget-garden

# Перейдите в папку backend
cd backend

# Установите зависимости (только при первом запуске)
npm install

# Запустите сервер
npm start

# Откройте сайт
нажмите http://localhost:3001
